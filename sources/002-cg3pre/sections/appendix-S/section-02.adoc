[[tocS_02]]
=== 地形の記述とLOD


[[tocS_02_01]]
==== 地形モデル

地形とは、地表の起伏の状態をいう。[<<gsi_ops,付録７ 公共測量標準図式>>]

標準製品仕様書では、地形の表現に必要な地物型等をCityGML及びi-URより抽出し、LODごとに、含むべき地物型、各地物型の空間属性の型、取得基準、取得方法及び補足を定めた「地形モデル」を定義する。

地形モデルのLODは、LOD0からLOD3までを対象とする。このとき、LODには外部・内部の区別はない。

[[fig-S-1]]
.地形モデルの概要
image::../../sources/images/anx_S_001.png[]

* 地形モデル（LOD0）は、地形の形状を、高さをもたない点又は線の集まりにより表現する。
* 地形モデル（LOD1）は、地形の形状を、高さをもつ三角形の集合又は高さをもつ点の集合により表現する。
* 地形モデル（LOD2）は、地形の形状を、高さをもつ三角形の集合又は高さをもつ点の集合により表現する。
* 地形モデル（LOD3）は、地形の形状を、高さをもつ三角形の集合又は高さをもつ点の集合により表現する。

LOD1からLOD3までは、作成に使用する原典資料の点密度または格子間隔により決定する。


[[tocS_02_02]]
==== 使用可能な地物型とLOD

地形モデルは、LODごとに使用すべき地物型やその空間属性を定めている。


[requirement]
.地形記述のLOD対応条件
====
[%metadata]
identifier:: /req/dem/1
subject:: 妥当な地形データセットの要件
description:: 地形の記述には、指定されたLODに対応する地物型及び空間属性を使用する。
====

地形は、dem:ReliefFeatureを用いて記述する。

地形はその表現方法に応じて地形要素（dem:AbstractReliefComponent）を選択する。 +
標準製品仕様書では、地形要素として、dem:TinRelief（不規則三角網）、dem:MassPointRlief（標高点）、dem:BreaklineRelief（等高線）及びdem:RasterRelief（標高グリッド）を地形モデルに含めている。地形の表現では、これら四つの地形要素から一つを選択し、その集まりとして地形を表現する。

地形の各LODにおいて使用可能な地物型とその空間属性を<<tab-S-1>>に示す。

// 地形モデルの記述に使用する地物型と空間属性
[[tab-S-1]]

3D都市モデルでは、地形モデルにdem:TINReliefを使用することを推奨している。この理由として、点群で扱う場合には、LAS形式など点群に特化したデータフォーマットが別途存在し、地形モデルには主題属性が付与されていないことから、CityGMLを使用して点群を記述する利点があまりないことが挙げられる。地形を連続した面として取り扱えるようにすることを重視し、dem:TINReliefの採用を推奨する。


[[tocS_02_03]]
==== 地形モデルのLOD定義

地形モデルは、LODごとに、含むべき地物やその取得基準を定めている。


[[tocS_02_03_01]]
===== LOD0

地形モデル（LOD0）は、地形要素として、等高線（dem:BreaklineRelief）、標高点（dem:MassPointRelief）又は標高グリッド（dem:RasterRelief）を選択し、その集まりにより表現する。

なお、公共測量標準図式に従った取得を行いたい場合は、数値地形図属性（urc:DmAttribute）を使用する。

[[tocS_02_03_02]]
===== LOD1

地形モデル（LOD1）は、地形要素として、不規則三角網（dem:TINRelief）、標高点（dem:MassPointRelief）又は標高グリッド（dem:RasterRelief）を選択し、その集まりにより表現する。 +
このとき、原典資料の点密度又は格子間隔は以下とする。

* レーザ点群の場合は、点密度0.04点/m2以上
* 数値標高モデルの場合は、点密度0.04点/m2以上のレーザ点群を使用して作成し、標高点格子間隔5m以内

[[tocS_02_03_03]]
===== LOD2

地形モデル（LOD2）は、地形要素として、不規則三角網（dem:TINRelief）、標高点（dem:MassPointRelief）又は標高グリッド（dem:RasterRelief）を選択し、その集まりにより表現する。 +
このとき、原典資料の点密度又は格子間隔により、LOD2.0及びLOD2.1に区分する。

* LOD2.0
** レーザ点群の場合は、点密度0.25点/m2以上
** 数値標高モデルの場合は、点密度0.25点/m2以上のレーザ点群を使用して作成し、標高点格子間隔2m以内
* LOD2.1
** レーザ点群の場合は、点密度1点/m2以上
** 数値標高モデルの場合は、点密度1点/m2以上のレーザ点群を使用して作成し、標高点格子間隔1m以内


[[tocS_02_03_04]]
===== LOD3

地形モデル（LOD3）は、地形要素として、不規則三角網（dem:TINRelief）、標高点（dem:MassPointRelief）又は標高グリッド（dem:RasterRelief）を選択し、その集まりにより表現する。 LOD3はLOD2よりも詳細な地形を表現する。
