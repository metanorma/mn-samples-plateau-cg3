[[tocN_02]]
=== その他の構造物の記述とLOD

[[tocN_02_01]]
==== その他の構造物モデル

その他の構造物とは、構造物のうち、建築物、橋梁及びトンネルを除く土木構造物をいう。「構造物」とは、「目的とする機能を持ち、作用に対して抵抗することを意図として人為的に構築されるもの」（土木・建築にかかる設計の基本、国土交通省）であり、ダム、堤防、床止め、堰（せき）、水門・閘門・陸閘、樋門・樋管、伏せ越し、水制、護岸、防波堤その他の土木構造物を指す。

その他の構造物の記述には、con:OtherConstructionを使用する。con:OtherConstructionは、CityGML3.0において新たに追加されたモジュールである構造物（Construction）モジュールの地物型である。

標準製品仕様書では、その他の構造物の表現に必要な地物型等をCityGML及びi-URから抽出し、LODごとに、含むべき地物型、各地物型の空間属性の型、取得基準、取得方法及び補足を定めた「その他の構造物モデル」を定義する。

その他の構造物モデルのLODは、LOD0からLOD3までを対象とする。このとき、その他の構造物モデルLODには外部しか存在しない。

[[fig-N-1]]
.その他の構造物モデルの概要
image::../../sources/images/anx_N_001.png[]

* LOD0は、その他の構造物の形状を、上方からの正射影の外周により囲まれた平面により表現する。
* LOD1は、その他の構造物の形状を、上方からの正射影の外周に一律の高さを与えて上向きに押し出した立体（箱モデル）により表現する。
* LOD2は、その他の構造物の形状を、屋根や壁の形状を含む立体又は多曲面（面の集まり）として表現し、立体の境界面又は多曲面を構成する面を屋根面、外壁面、底面等に区分する。
* LOD3は、その他の構造物の形状を、構造部材の外形を表す面により構成された立体又は多曲面（面の集まり）として表現する。

その他の構造物は、橋梁（brid:Bridge）やトンネル（tun:Tunnel）と同様に、本体（con:OtherConstruction）とその付属物（uro:ConstructionInstallation）から構成される。 +
また、本体の外形は、境界面（con:AbstractConstructionSurface）に区分される。境界面はその向きや機能により、屋根面（con:RoofSurface）、外壁面（con:WallSurface）、底面（con:GroundSurface）、閉鎖面（core:ClosureSurface）、屋外床面（con:OuterFloorSurface）及び屋外天井面（con:OuterRoofSurface）に区分する。

* LOD0では、con:OtherConstructionを使用し、構造物の幾何形状を平面で記述する。
* LOD1では、con:OtherConstructionを使用し、構造物の幾何形状を立体で記述する。
* LOD2では、con:OtherConstructionを使用し、構造物の幾何形状を立体で記述する。また、立体を構成する境界面を、屋根面や外壁面等の地物型を用いて記述する。屋根面や外壁面はcon:AbstractConstructionSurfaceとして定義されている。con:AbstractConstructionSurfaceは抽象地物であり、実装にはこれを継承する地物型（例：屋根面はcon:RoofSurface）を使用する。また、扉面（con:DoorSurface）や窓面（con:WindowSurface）を用いることもできる。付属物（uro:OtherConstructionInstallation）を表現することもできる。
* LOD3では、con:OtherConstructionを使用し、建築物の幾何形状を多曲面で記述する。また、構造物は構造部材（uro:OtherConstructiveElement）の集まりとして構成され、構造物の多曲面を構成する面は、構造部材を構成する面に一致する。また、扉（con:Door）及び窓（con:Window）を含めることができる。付属物（uro:OtherConstructionInstallation）を表現することもできる。

[[tocN_02_03]]
==== その他の構造物モデルのLOD定義

その他の構造物モデルは、LODごとに、含むべき地物やその取得基準を定めている。


[[tocN_02_03_01]]
===== LOD0

その他の構造物モデル（LOD0）は、含むべき地物によりLOD0.0及びLOD0.1に細分される。

LOD0.0はその他構造物の正射影の外周を面により表現する。 +
LOD0.1は、接地面の正射影の外周か屋根面の正射影の外周かを区別して取得する。

標準製品仕様書では、原則としてLOD0.0を採用する。ただし、ユースケースの必要に応じてLOD0.1を採用できる。

[[tocN_02_03_02]]
===== LOD1

その他の構造物モデル（LOD1）は、含むべき地物によりLOD1.0及びLOD1.1に細分される。

その他の構造物のLOD1は、core:lod0MultiSurfaceが示すその他の構造物の外周を示す平面（又はcore:lod0MultiCurveが示すその他の構造物の外周を閉じて作成した平面）を、一定の高さで上向きに押し出した角柱として表現したものである。 +
LOD1.1は、LOD1.0の角柱の境界面を、地物に区分することができる。

標準製品仕様書は、原則としてLOD1.0を採用する。ただし、ユースケースの必要に応じてLOD1.1を採用できる。

[[tocN_02_03_03]]
===== LOD2

その他の構造物モデル（LOD2）は、LOD2.0及びLOD2.1に細分される。 +
その他の構造物モデル（LOD2）は、測量による取得を前提とした区分である。

LOD2.0は、その他構造物の形状を上面から詳細化した立体として表現する。立体の境界面は、屋根面（con:RoofSurface）、外壁面（con:WallSurface）及び底面（con:GroundSurface）に区分する。また、行政界など仮想的な面で構造物を区切る場合には閉鎖面（con:ClosureSurface）を用いる。 +
LOD2.1では、その他構造物の形状を上面及び側面から詳細化した立体として表現する。立体の境界面は、屋根面（con:RoofSurface）、外壁面（con:WallSurface）及び底面（con:GroundSurface）に区分する。 +
また、行政界など仮想的な面で構造物を区切る場合には閉鎖面（core:ClosureSurface）を用いる。 +
さらに、ユースケースの必要に応じて、屋外天井面（con:OuterCeilingSurface）、屋外床面（con:OuterFloorSurface）、及び開口部の境界面（con:DoorSurface及びcon:WindowSurface）を使用することができる。 +
また、付属物（con:OtherConstructionInstallation）を表現することもできる。

標準製品仕様書は、原則としてLOD2.0を採用する。ただし、ユースケースの必要に応じてLOD2.1を採用できる。

[[tocN_02_03_04]]
===== LOD3

その他の構造物モデル（LOD3）は、含むべき地物によりLOD3.0及びLOD3.1に細分される。 +
その他の構造物モデル（LOD3）は、CIMモデルやCADデータからの変換による作成を前提とした区分である。

LOD3.0は、その他構造物を構造部材（uro:OtherConstructiveElement）の集まりとして表現し、その他構造物の形状を、構造部材を構成する面を集めた多曲面として表現する。 +
LOD3.1は、LOD3.0におけるその他構造物の幾何形状を構成する各面を、境界面を表す地物に区分することができる。

標準製品仕様書は、原則としてLOD3.0を採用する。ただし、ユースケースの必要に応じてLOD3.1を採用できる。
