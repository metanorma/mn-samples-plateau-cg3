[[tocT_02]]
=== 水部の記述とLOD


[[tocT_02_01]]
==== 水部モデル

水部とは、河川、細流、一条河川、かれ川、用水路、湖池、海岸線、水路地下部及び低位水涯線をいう。[<<gsi_ops,付録７ 公共測量標準図式>>]

標準製品仕様書では、水部の表現に必要な地物型等をCityGML及びi-URより抽出し、LODごとに、含むべき地物型、各地物型の空間属性の型、取得基準、取得方法及び補足を定めた「水部モデル」を定義する。

水部モデルのLODは、LOD0からLOD3までを対象とする。このとき、水部に外部・内部の区別はない。

[[fig-T-1]]
.水部モデルの概要
image::../../sources/images/anx_T_001.png[]

* 水部モデル（LOD0）では、水部の形状を、線により表現する。水部の形状の線は、河川ネットワーク又は水涯線のいずれかとする。
* 水部モデル（LOD1）では、水部の形状を、高さをもった面として表現する。
* 水部モデル（LOD2）では、水部の形状を立体として表現する。水部の立体は、水涯線により囲まれた面（水面）、水底面及び管理区間や行政区域を区切る面を境界面とする。また、水部の立体は、水中にある構造物が占める空間を除いてもよい。
* 水部モデル（LOD3）では、水部の形状を立体として表現する。このとき、水涯線により囲まれた面（水面）、及び水底面は、水部モデル（LOD2）よりも詳細な形状となる。


[[tocT_02_02]]
==== 使用可能な地物型とLOD

水部モデルは、LODごとに、使用すべき地物型やその空間属性が異なる。


[requirement]
.水部記述のLOD対応条件
====
[%metadata]
identifier:: /req/wtr/1
subject:: 妥当な水部データセットの要件
description:: 水部の記述には、指定されたLODに対応する地物型及び空間属性を使用する。
====

水部は、wtr:WaterBodyを用いて記述する。 +
水部（wtr:WaterBody）、この境界面となる水部の境界面（wtr:AbstractBoundarySurface）及び仮想的な境界面となる閉鎖面（core:ClosureSurface）はLODに応じた空間属性をもつ。が定義される。水部の境界面は、水面（wtr:WaterSurface）、水底面（wtr:WaterGroundSurface）及び閉鎖面（core: ClosureSurface）に区分される。

* LOD0では、wtr:Waterbodyを使用し、水部の形状を線として表現する。
* LOD1では、wtr:WaterBodyを使用し、水部の形状は、その境界面となる水面（wtr:WaterSurface）を用いて面として表現する。
* LOD2では、wtr:WaterBodyを使用し、水部の形状を立体として表現する。また、その立体の境界面をwtr:WaterSurface、wtr:WaterGroundSurface又はcore:ClosureSurfaceに区分する。
* LOD3では、LOD2と同様に、wtr:WaterBodyを使用して水部の形状を立体として表現し、その立体の境界面をwtr:WaterSurface、wtr:WaterGroundSurface又はcore:ClosureSurfaceに区分する。ただし、その形状はLOD2より詳細化される。

==== 水部モデルのLOD定義

水部モデルは、LODごとに含むべき地物やその取得基準を定めている。

[[tocT_02_03_01]]
===== LOD0

水部モデル（LOD0）は、河心線（現況河道の中心となる線）又は水涯線により表現する。

[[tocT_02_03_02]]
===== LOD1

水部モデル（LOD1）は、水部の形状を、高さをもった水面により表す。 +
このとき水部（wtr:WaterBody）の空間属性は空となり、水部の形状は境界面となる水面（wtr:WaterSurface）の面により水部の形状を表現する。

// 水部モデル（LOD1）の取得イメージ
[[tab-T-3]]
include::../../../common/sections/com_wtr_Acquire_lod1.adoc[]


[[tocT_02_03_03]]
===== LOD2

水部モデル（LOD2）は、水部を水面や水底面を境界面とする立体により表現する。

水部モデル（LOD2）は、水中の構造物の表現有無により、LOD2.0とLOD2.1に分かれる。 +
LOD2.0の場合、水部の立体は、水中に存在する構造物が占有する空間も含めた空間となる。 +
LOD2.1の場合、水部の立体は、水中に存在する構造物が占有する空間を除いた空間となる。

[[tocT_02_03_04]]
===== LOD3

水部モデル（LOD3）は、水部モデル（LOD2）と同様に、水面や水底面を境界面とする立体により表現する。 +
水部モデル（LOD2）は、測量により取得したデータを用いて作成することを前提としているが、水部モデル（LOD3）は、CIMモデルなどを用いて作成することを前提としている（例：運河）。

水部モデル（LOD3）は、水中の構造物の表現有無により、LOD3.0とLOD3.1に分かれる。 +
LOD3.0の場合、水部の立体は、水中に存在する構造物が占有する空間も含めた空間となる。 +
LOD3.1の場合、水部の立体は、水中に存在する構造物が占有する空間を除いた空間となる。
