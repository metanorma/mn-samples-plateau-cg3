[[tocL_02]]
=== 橋梁の記述とLOD

[[tocL_02_01]]
==== 橋梁モデル

橋梁とは、道路、鉄道、水路等の輸送路において、輸送の障害となる河川、渓谷、湖沼、海峡あるいは他の道路、鉄道、水路等の上方にこれらを横断するために建設される構造物の総称である。市街地において効率的な土地利用の観点から、道路上あるいは河川上の空間に連続して建設される高架橋も橋梁の一形態である。[土木工学ハンドブック] +
標準製品仕様書では以下を対象とする。

* 道路法（昭和27年法律第180号）第2条第1項に規定する「橋」。橋長2.0m以上を対象とし、高架橋及び桟道橋を含む[参考：中部地方整備局　道路施設台帳作成マニュアル]。
* 鉄道事業者法施行規則別表第一に定める鉄道施設の「橋りょう」。
* 鉄道事業者法施行規則別表第一に定める鉄道施設の「こ線橋」。
* 道路法第30条第1項第10号に定める「横断歩道橋」。

また、標準製品仕様書では、建築基準法第44条第1項第4号において示された公共用歩廊で、道路上に設けられた高架の歩行者専用道路（ペデストリアンデッキ、スカイウェイ、スカイウォークなどと呼ばれる）を含む。

標準製品仕様書では、橋梁の表現に必要な地物型等をCityGML及びi-URから抽出し、LODごとに、含むべき地物型、各地物型の空間属性の型、取得基準、取得方法及び補足を定めた「橋梁モデル」を定義する。

橋梁モデルのLODは、LOD0からLOD3までを対象とする。このとき、LODには外部（屋外）と内部（屋内）がそれぞれ存在する。

[[fig-L-1]]
.橋梁モデルの概要
image::../../sources/images/anx_L_001.png[]

* 橋梁モデル・外部（LOD0）は、橋梁の形状を、外周を表す平面により表現する。
* 橋梁モデル・外部（LOD1）は、橋梁の形状を、外周を表す平面を一律の高さで上向きに押し出した立体（箱モデル）により表現する。
* 橋梁モデル・外部（LOD2）は、橋梁の形状を、屋根や壁の形状を含む立体又は多曲面（面の集まり）として表現し、立体の境界面又は多曲面を構成する面を屋根面、外壁面、底面等に区分する。
* 橋梁モデル・外部（LOD3）は、橋梁の形状を、構造部材の外形を表す面により構成された立体又は多曲面として表現する。また外部の付属物を表現する。
* 橋梁モデル・内部（LOD0）は、橋梁内部に存在する空間を、その外周を表す平面又は線により表現する。
* 橋梁モデル・内部（LOD1）は、橋梁内部に存在する空間を、その外周を表す平面に一律の高さを与えて押し出した立体（箱モデル）により表現する。
* 橋梁モデル・内部（LOD2）は、橋梁内部に存在する空間を、床や天井の形状を含む立体として表現し、その境界面を天井面、内壁面、床面等に区分する。
* 橋梁モデル・内部（LOD3）は、橋梁内部に存在する空間を床や天井の形状を含む立体として表現するとともに、構造部材の外形を多曲面として表現する。


[[tocL_02_02]]
==== 使用可能な地物型とLOD

橋梁モデルは、LODごとに、使用すべき地物型やその空間属性が異なる。


[requirement]
.橋梁記述のLOD対応条件
====
[%metadata]
identifier:: /req/brid/1
subject:: 妥当な橋梁データセットの要件
description:: 橋梁の記述には、指定されたLODに対応する地物型及び空間属性を使用する。
====

橋梁は、brid:Bridgeを用いて記述する。 +
橋梁、これを構成する屋根及び壁、開口部（窓や扉）、付属物、内部空間、構造部材等は、LODに応じた空間属性をもつ。

* 外部LOD
** 外部（LOD0）では、brid:Bridgeを使用し、橋梁の幾何形状を平面で記述する。
** 外部（LOD1）では、brid:Bridgeを使用し、橋梁の幾何形状を立体で記述する。
** 外部（LOD2）では、brid:Bridgeを使用し、橋梁の幾何形状を立体又は多曲面で記述する。また、立体を構成する境界面又は多曲面を構成する面を、屋根面や外壁面等の地物型を用いて記述する。屋根面や外壁面はcon:AbstractConstructionSurfaceとして定義されている。con:AbstractConstructionSurfaceは抽象地物であり、実装にはこれを継承する地物型（例：屋根面はcon:RoofSurface）を使用する。また、扉面（con:DoorSurface）や窓面（con:WindowSurface）を用いることもできる。
** 外部（LOD3）では、brid:Bridgeを使用し、橋梁の幾何形状を立体又は多曲面で記述する。また、橋梁は構造部材（brid:BridgeConstructiveElement）の集まりとして構成され、橋梁の立体を構成する境界面又は多曲面を構成する面は、構造部材を構成する面に一致する。また、扉（con:Door）及び窓（con:Window）を含めることができる。
* 内部LOD
** 内部（LOD0）では、brid:Bridgeを使用し、橋梁を内部空間（brid:BridgeRoom）の集まりとして記述する。部屋の幾何形状は、平面又は線で記述する。
** 内部（LOD1）では、brid:Bridgeを使用し、橋梁を内部空間（brid:BridgeRoom）の集まりとして記述する。部屋の幾何形状は、立体で記述する。
** 内部（LOD2）では、brid:Bridgeを使用し、橋梁を内部空間（brid:BridgeRoom）の集まりとして記述する。部屋の幾何形状は、立体で記述する。また、立体の境界面を構成する面を、天井面や内壁等の地物型を用いて記述する。天井面や内壁面はcon:AbstractConstructionSurfaceとして定義されている。con:AbstractConstructionSurfaceは抽象地物であり、実装にはこれを継承する地物型（例：天井面はcon:CeilingSurface）を使用する。また、扉面（con:DoorSurface）や窓面（con:WindowSurface）を用いることもできる。
** 内部（LOD3）では、brid:Bridgeを使用し、橋梁を内部空間（brid:BridgeRoom）及び構造部材（brid:BridgeConstructiveElement）の集まりとして記述する。また、扉（con:Door）及び窓（con:Window）を含めることができる。内部空間の幾何形状は立体、建築部材、扉及び窓の幾何形状は多曲面で記述する。

各LODでは、brid:BridgePartを使用できる。brid:BridgePartは、一つの橋梁を二つ以上の部分に分けた橋梁の一部であり、brid:Bridgeと同様の属性をもつことができる。例えば、一つの橋梁について、径間ごとに構造が異なっていたり、作成に使用する原典資料や作成手法が異なっていたりする場合に、その橋梁を径間ごとに区切り、区切った橋梁の部分（brid:BridgePart）ごとに属性を与えるために使用する。

ただし、このとき、[underline]#brid:Bridgeを構成するbrid:BridgePart同士は互いに接して一連の橋梁を構成しなければならない。#

また、各LODでは橋梁に付属する階段やスロープといった設備を、brid:BridgeInstallationを用いて記述できる。

なお、brid:BridgePartは橋梁であるが、brid:BridgeInstallationは橋梁ではなく、橋梁の付属物である。外部LODで使用する場合、brid:BridgeInstallationは橋梁の外側に付く付属物であり、内部LODで使用する場合は内部空間の付属物となる。

さらに、内部LODでは、可動設備（brid:BridgeFurniture）を含めることができる。

[NOTE,type=commentary]
--
LOD0は、数値地形図との互換性を考慮した区分である。LOD1は、LOD0の編集を前提とした区分である。また、LOD2は、航空写真等上空からの取得やMMSによる点群や画像等、側面からの取得を前提とした区分である。さらに、LOD3は、BIM/CIM又は図面の利用を前提とした区分である。 +
ただし橋梁の形状の一部は、橋台や橋脚のように、道路下部に存在し、上空からの取得が困難である。そこで、LOD2は、原典資料から取得できない部分の形状の推定有無によりLODを細分している。
--

==== 橋梁モデルのLOD定義

橋梁モデルは、LODごとに、含むべき地物やその取得基準を定めている。


[[tocL_02_03_01]]
===== 外部LOD0

橋梁モデル・外部（LOD0）は、含むべき地物により、LOD0.0及びLOD0.1に細分される。

LOD0.0は地図情報レベル2500の数値地形図の利用を前提とした区分であり、橋梁の外周を面により表現する。 +
LOD0.1は、接地面の正射影の外周か屋根面の正射影の外周かを区別して取得する。

標準製品仕様書では、原則としてLOD0.0を採用する。ただし、ユースケースの必要に応じてLOD0.1を採用できる。

[[tocL_02_03_02]]
===== 外部LOD1

橋梁モデル・外部（LOD1）は、含むべき地物によりLOD1.0及びLOD1.1に細分される。

LOD1.0は、LOD0で作成したcore:lod0MultiSurfaceが示す橋梁の外周の平面を、一定の高さで上向きに押し出した角柱として表現したものである。 +
LOD1.1は、LOD1.0の立体を構成する境界面を地物に区分する。

標準製品仕様書は、原則としてLOD1.0を採用する。ただし、ユースケースの必要に応じてLOD1.1を採用できる。

[[tocL_02_03_03]]
===== 外部LOD2

橋梁モデル・外部（LOD2）は、含むべき地物やその大きさによりLOD2.0及びLOD2.1に細分される。橋梁モデル・外部（LOD2）は、測量による取得を前提とした区分である。

LOD2.0は、橋梁の形状を上面から詳細化した多曲面（面の集まり）として表現する。橋梁を構成する面は、屋外床面（con:OuterFloorSurface）とする。 +
LOD2.1では、橋梁の形状を上面及び側面から詳細化した立体として表現する。立体の境界面は、屋根面（con:RoofSurface）、外壁面（con:WallSurface）及び底面（con:GroundSurface）に区分する。 +
さらに、ユースケースの必要に応じて、屋外天井面（con:OuterCeilingSurface）、屋外床面（con:OuterFloorSurface）、及び開口部の境界面（con:DoorSurface及びcon:WindowSurface）を使用することができる。また、付属物（brid:BridgeInstallation）や主要な構造部材（brid:BridgeConstructiveElement）を表現することもできる。

標準製品仕様書は、原則としてLOD2.0を採用する。ただし、ユースケースの必要に応じてLOD2.1を採用できる。

===== 外部LOD3

橋梁モデル・外部（LOD3）は、含むべき地物によりLOD3.0及びLOD3.1に細分される。橋梁モデル（LOD3）は、CIMモデルやCADデータからの変換による作成を前提とした区分である。

LOD3.0は、橋梁を構造部材（brid:BridgeConstructiveElement）の集まりとして表現し、橋梁の形状を、構造部材を構成する面を集めた多曲面として表現する。 +
LOD3.1は、LOD3.0における橋梁の幾何形状を構成する各面を、境界面を表す地物に区分することができる。

標準製品仕様書は、原則としてLOD3.0を採用する。ただし、ユースケースの必要に応じてLOD3.1を採用できる。

===== 内部LOD1

橋梁モデル・内部（LOD1）は、含むべき地物によりLOD1.0及びLOD1.1に細分される。

LOD1.0は、core:lod0MultiSurfaceが示す橋梁の内部の外周を示す平面（又はcore:lod0MultiCurveが示す橋梁の内部の外周を閉じて作成した平面）を、一定の高さで上向きに押し出した角柱として表現したものである。 +
