== はじめに

* Project PLATEAUは2020年に始動し、3D都市モデルの流通性・再利用性を高め、その整備・活用・オープンデータ化を推進している。PLATEAUでは、地理空間データに関する標準化団体であるOpen Geospatial Consortium（OGC）が策定した国際標準「CityGML」と、その拡張規則であるApplication Domain Extension（ADE）に基づき、内閣府地方創生推進事務局が都市再生に必要なデータを拡張した「i-都市再生技術仕様（案）」（i-UR）を採用し、3D都市モデルの「標準製品仕様」を示す。 

* 標準製品仕様はCityGML 2.0に準拠しているが、2023年に最新の国際標準であるCityGML 3.0が発行された。そこで、PLATEAUでは、CityGML 3.0への移行に向けた検討を進めている。本書は、2025年度の検討成果として、標準製品仕様の主要な要素である、CityGML 3.0及びこれに基づくi-URに準拠した「応用スキーマ（UMLクラス図とその定義文書）」の案と解説を示すものである。 

* CityGML 3.0では、従来の3D都市モデルのユースケースだけではなく、将来的に想定される多様なユースケースに適用できるよう、データ構造の柔軟性が高められている。しかし、選択肢の増加は、標準製品仕様の利用者に混乱をもたらす可能性がある。また、多様な選択肢に対応した3D都市モデルの作成や利用ツールの開発は非効率となることが懸念される。 

* このため、PLATEAUでは、標準製品仕様の網羅性を確保しつつ、実用性の高い3D都市モデルが整備・活用されるように、本書で示す応用スキーマ案を踏まえ、推奨する選択肢やその優先順位を示したプロファイルを検討する予定である。 

* 本書を公開することにより、標準製品仕様の利用者である3D都市モデルの整備事業者や利用者の理解を促進し、円滑なCityGML 3.0への移行を目指す。また、応用スキーマの案に対するフィードバックを反映させることで、実用的な標準製品仕様の確立を目指す。
