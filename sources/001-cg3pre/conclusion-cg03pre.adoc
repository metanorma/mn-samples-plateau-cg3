== おわりに 

* 本書では、Project PLATEAUにおいて、2028年度からCityGML 3.0準拠の3D都市モデル整備・活用・オープンデータ化を本格運用することを目指し、標準製品仕様の主要な要素である、CityGML 3.0及びこれに基づくi-URに準拠した「応用スキーマ」の案を技術文書として示した。 

* CityGML 3.0では、屋内におけるデータ記述の多様化、3D都市モデルとセンサーデータや点群データ等との統合、BIMを含む他分野の標準との整合性向上など、多岐にわたる改良が加えられている。また、従来の3D都市モデルのユースケースだけではなく、将来的に想定される多様なユースケースに適用できるよう、データ構造の柔軟性が高められている。今後は、CityGML 3.0を採用する国や都市が増えていくと想定される。 

* しかし、多様性や柔軟性が高まることに伴い、各都市で作成される3D都市モデルが多様化し、再利用性が低下する懸念も残っている。 

* このため、PLATEAUでは、標準製品仕様の網羅性を確保しつつ、実用性の高い3D都市モデルが整備・活用されるように、推奨する選択肢やその優先順位を示したプロファイルを標準作業手順として策定する予定である。プロファイルの作成は、2026年度及び2027年度に実施されるCityGML 3.0準拠の3D都市モデル整備実証や、標準製品仕様の利用者からのフィードバックを基に行い、これにより円滑なCityGML 3.0への移行を目指している。
