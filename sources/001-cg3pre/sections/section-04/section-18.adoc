=== 地形モデルの応用スキーマ

地形とは、地表の起伏の状態をいう。[<<gsi_ops,付録７ 公共測量標準図式>>]

[[toc4_18_01]]
==== 地形モデルのLOD

標準製品仕様書が対象とする地形モデルのLODは、LOD0からLOD3までとする。

[[toc4_18_01_01]]
===== 地形モデル（LOD0）

[[toc4_18_01_01_01]]
====== 地形モデル（LOD0）の概要

地形モデル（LOD0）では、地形を点又は線により表現する。 +
ただし、数値地形図との互換性を保つ必要がある場合は、公共測量標準図式に従った<<toc4_25_01_01,公共測量標準図式のためのLOD0の定義>>を採用できる。

[[toc4_18_01_03]]
===== 地形モデル（LOD2）

[[toc4_18_01_03_01]]
====== 地形モデル（LOD2）の概要

地形モデル（LOD2）では、地形を、標高をもつ点の集まり、標高を属性としてもつ正方格子（ラスター）又は標高をもつ点から構成する三角形の集合により表現する。 +
なお、格子間隔の詳細度により、LOD2.0及びLOD2.1に区分する。 +
LOD2.0は、格子間隔2m以内（地図情報レベル2500）とし、LOD2.1は、格子間隔1m以内（地図情報レベル1000）としている。
