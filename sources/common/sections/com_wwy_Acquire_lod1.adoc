h| 説明
2+| 法令により指定された航路の区域を面として取得する。高さは0とする。 +
このとき、属性granularityによって、この航路の区域全体又は進行方向単位なのか、あるいは航路単位なのかを表現する。 +
granularityがwayかつ航路の区域全体として表現する場合は、航路の区域を表現する。 +
granularityがwayかつ航路の進行方向単位として表現する場合は、航路の境界線をつないだ面を取得し、進行方向単位の航路を表現する。 +
granularity=laneの場合は、航路の境界線をつないだ面を取得し、各航路を表現する。 +
　 +
交通空間（TrafficSpace）はTrafficAreaを底面とし、底面から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは、船舶の航行を妨げ得る上空障害（橋梁桁下、架空線、門型クレーン等）によって規定される最小可航高とする。 +
また、離隔空間（ClearanceSpace）はTrafficSpaceの上面を底面とし、底面から一律の高さで上向きに押し出した立体を作成する。このとき、一律の高さは、無人航空機が飛行可能な高さを原則とする。
