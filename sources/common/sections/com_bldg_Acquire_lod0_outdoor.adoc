h| 説明
| 建築物の正射影の外周を取得する。
| 建築物の正射影の外周か接地面の正射影の外周のどちらかを取得する。 +
LOD0.0は建築物の正射影の外周を取得するが、LOD0.1では建築物の正射影の外周か接地面の正射影の外周かを区別して取得する。建築物の正射影の外周の場合は、RoofSurfaceに空間属性を持たせ、接地面の場合はGroundSurfaceに空間属性を持たせることで区分する。また、外周を構成する線を区分できる。 +
取得例の場合は、外壁及び扉に区分している。
