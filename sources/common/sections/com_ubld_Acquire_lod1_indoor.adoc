h| 説明
| 床面の上方からの正射影の外周を取得し、床面から一律の高さで上向きに押し出した立体を作成する。
| 地下街モデル・内部（LOD1.0）の形状から、その立体を構成する面の全部又は一部を境界面として区分することができる。 +
このとき、少なくとも一つの面は天井面、内壁面、床面、窓、ドア等の境界面として区分されなければならない。 +
なお、境界面への区分は幾何形状の詳細化を伴わず、LOD1.0と同等の単純押出形状を保持するものとする。
