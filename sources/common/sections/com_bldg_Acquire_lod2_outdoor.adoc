.建築物モデル・外部（LOD2）の取得イメージ
|===
h| LOD ^h| 取得例 ^h| 説明
h| LOD2.0 
^.^| 
image::../../sources/images/bldg_Acquire_lod20_out.png[]
| 屋根の主要な外形が再現される。 +
必要に応じて付属物、開口部及び軒の取得を行うことができる。 +
付属物を取得する場合は、短辺の実長3m以上又は短辺の実長1m以上かつ上方又は側方からの正射影の面積3m2以上の規模の大きいものを対象とする。 +
開口部を取得する場合は、短辺の実長1m以上のものを対象とする。 +
軒を取得する場合は、幅3m以上の軒裏を対象とする。 +
取得例では、以下の3つを示す。 +

* 「屋根の主要な外形を再現したもの」
* 「付属物及び開口部を取得したもの」
* 「付属物、開口部及び軒裏を取得したもの」

なお、付属物を取得しない場合、バルコニーは屋根として取得される。 +
また、軒を取得しない場合、バルコニーの下部も建築物の一部として表現される。
