^.^|
image::../../sources/images/trk_Acquire_lod30.png[]

h| 説明
| 徒歩道を構成する各部分を区別して外形を三次元的に表現し、徒歩道の形状及び空間的広がりが判別可能な程度の面の集まりとして表現する。 +
表現対象は、既存の三次元モデルに含まれる情報の範囲に応じて選択するものとし、モデルに含まれない要素は省略してよい。 +
なお、各部分はユースケースの必要に応じて細分して表現できる。
