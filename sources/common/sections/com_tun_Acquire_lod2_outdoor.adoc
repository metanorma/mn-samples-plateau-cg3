.トンネルモデル・外部（LOD2）の取得イメージ
|===

h| 説明 
|
トンネルの外形を立体として表現し、立体の各境界面を、屋根や外壁に区分する。 +
開口部の再現有無及び付属物の再現有無は、属性granuralityにて表現する。 +
上図は開口部の再現あり・付属物の再現ありの場合であり、トンネルの外側に付いている付属物として、坑門や連絡坑を取得する。
|
地下横断歩道の外形を立体として取得し、立体の各境界面を、屋根や外壁に区分する。 +
開口部の再現有無及び付属物の再現有無は、属性granuralityにて表現する。 +
上図は付属物の再現なしの場合であり、地下横断歩道の出入口に設けられた建屋は、付属物ではなく、都市設備（CityFurniture）として取得する。
