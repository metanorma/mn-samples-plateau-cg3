h| 説明
| 地下街の上からの正射影の外周を、地表面から地下街の下端まで下向きに押し出した立体として表現する。
| 地下街モデル・外部（LOD1.0）の形状から、その立体を構成する面の全部又は一部を境界面として区分することができる。 +
このとき、少なくとも一つの面は屋根面、外壁面、底面、窓面、扉面等の境界面として区分されなければならない。 +
なお、境界面への区分は幾何形状の詳細化を伴わず、LOD1.0と同等の単純押出形状を保持するものとする。
