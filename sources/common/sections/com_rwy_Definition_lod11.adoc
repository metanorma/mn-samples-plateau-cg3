【granularityがwayの場合の取得方法】

* 建築限界の上面を底面とし、無人航空機が飛行可能な範囲を立体として作成する。
|一律の高さは、無人航空機が飛行可能な高さを原則とする。

|LOD1.1
|●
|TrafficArea
|lod1MultiSurface
|GM_MultiSurface
|普通鉄道、地下鉄地上部、路面鉄道、特殊鉄道、地下鉄地上部、路面鉄道、特殊鉄道、トンネル内の鉄道、地下鉄地下部
|
【granularityがwayの場合の取得方法】

* 左右のレールに囲まれた範囲を取得する。

【granularityがlaneの取得方法】

* 左右のレールに囲まれた範囲を取得する。
* 高さは0とする。
* 軌道ごとに一つの地物とする。
|
