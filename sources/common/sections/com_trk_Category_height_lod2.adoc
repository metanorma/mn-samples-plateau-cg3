.交通（徒歩道）モデルにおけるLOD2.0、LOD2.1及びLOD2.2の「高さの取得方法」
|===
| 取得方法 ^| LOD2.0 ^| LOD2.1 ^| LOD2.2

| 徒歩道の横断方向の高さは一律とし、車道の高さとする。 ^| ● ^| 　 ^| 　
| 徒歩道の横断方向に15㎝以上の高さの差が存在した場合に、車道部、歩道部、島それぞれの高さを取得する。 ^| 　 ^| ● ^| 　
| 徒歩道の横断方向に2㎝以上の高さの差が存在した場合に、車道部、歩道部、島それぞれの高さを取得する。 ^| 　 ^| 　 ^| ● footnoteblock:[fn_track_height_usecase]

|===

■:: 条件付必須
〇:: 任意（ユースケースに応じて要否を決定してよい）
