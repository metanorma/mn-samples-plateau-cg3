h| 説明
| トンネル内部に存在する上向きの面の正射影の外周を取得する。
| トンネル内部に存在する上向きの面の正射影の外周かトンネル内部に存在する下向きの面の正射影の外周のどちらかを取得する。 +
LOD0.0は上向きの面の正射影の外周を取得するが、LOD0.1では上向きの面の正射影の外周か下向きの面の正射影の外周かを区別して取得する。上向き面の場合は、FloorSurfaceに空間属性を持たせ、下向きの面の場合はCeilingSurfaceに空間属性を持たせることで区分する。
