【granularityがlaneの場合の取得方法】

* 15㎝以上の段の場合は、段の上端と下端を結ぶ面を作成し、その形状を取得する。

* 15㎝未満の段が複数存在する場合は、最下段の下端と最上段の上端を結ぶ面を作成する。
* 高さの差を表現する面は、歩道部の一部として取得する。

【granularityがwayかつ道路全幅として表現する場合の取得方法】

【granularityがlaneの場合の取得方法】
