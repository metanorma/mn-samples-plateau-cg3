|===
|LOD|必須|地物型|関連役割名|空間属性の型|取得基準|取得方法|補足

|LOD2.0
|●
|CityFurniture
|lod2MultiSurface
|GM_MultiSurface
|他の地物の面と一体として表現されるべき地物
|
* 都市設備の外周を取得する。
* 外周の各頂点に、都市設備が接している他の地物の高さを与える。
|
* 外周の各頂点に与える高さとは、例えば、路面標示やマンホールの場合は、路面の高さとなる。
* 連続する面の集まりの場合は、GM_CompositeSurfaceを使用できる。

|LOD2.0
|●
|CityFurniture
|lod2Solid
|GM_Solid
|独立して存在する地物
|
* 都市設備の外周の形状が変化する高さで、各外周を取得する。
* 外周の各頂点を結ぶ立体を作成する。
|外周は、データセットが採用する地図情報レベルの水平及び高さの誤差の標準偏差に収まるよう取得する。

|LOD2.0
|●
|CityFurniture
|lod2MultiSurface
|GM_MultiSurface
|独立して存在する地物
|
* 都市設備の外周の形状が変化する高さで、各外周を取得する。
* 外周の各頂点を結ぶ面の集まりを作成する。
|外周は、データセットが採用する地図情報レベルの水平及び高さの誤差の標準偏差に収まるよう取得する。

|===
