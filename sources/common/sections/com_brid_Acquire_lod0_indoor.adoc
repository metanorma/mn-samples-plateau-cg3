h| 説明
| 床面の正射影の外周を取得する。
| 床面の正射影の外周か天井面の正射影の外周のどちらかを取得する。 +
LOD0.0は床面の正射影の外周を取得するが、LOD0.1では床面の正射影の外周か天井面の正射影の外周かを区別して取得する。床面の場合は、FloorSurfaceに空間属性を持たせ、天井面の場合はCeilingSurfaceに空間属性を持たせることで区分する。
