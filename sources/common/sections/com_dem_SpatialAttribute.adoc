ReliefFeatureを作成する場合は、LODに応じて以下のいずれかを必須とする。

* LOD0：MassPointRelief、BreaklineReliefまたはRasterReliefのいずれか。
* LOD1、LOD2及びLOD3：TinRelief、MassPointRelief又はRasterReliefのいずれか。
