|【granularityがlaneの場合の取得方法】

* TrafficAreaを底面とし、底面から一律の高さで上向きに押し出した立体を作成する。

* 底面の高さは0とする。
|一律の高さは、船舶の航行を妨げ得る上空障害（橋梁桁下、架空線、門型クレーン等）によって規定される最小可航高とする。
