h| 説明
| 地下街の正射影の外周を取得する。 +
高さは0とする。
| 地下街の正射影の外周か接地面の正射影の外周のどちらかを取得する。 +
LOD0.0は地下街の正射影の外周を取得するが、LOD0.1では地下街の正射影の外周か接地面の正射影の外周かを区別して取得する。地下街の正射影の外周の場合は、RoofSurfaceに空間属性を持たせ、接地面の場合はGroundSurfaceに空間属性を持たせることで区分する。
