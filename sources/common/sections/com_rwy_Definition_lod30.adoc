|LOD3
|●
|TrafficArea
|lod3Network
|GM_Complex
|
|
【granularityがlaneの場合の取得方法】

* 軌道中心線を取得する。
* 勾配変化点の標高を用いて軌道中心線を構成する各頂点に高さを与える。

|

|LOD3
|●
|TrafficArea
|lod3MultiSurface
|GM_MultiSurface
|
|【granularityがlaneの場合の取得方法】

* 外周を取得し、勾配が変化する場所で区切る。
* 外周の各頂点に、水平位置に対応する標高を与える。

【granularityがwayの場合の取得方法】

* 外周を取得し、勾配が変化する場所で区切る。
