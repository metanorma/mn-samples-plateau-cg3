[NOTE]
--
マンホール本体の外周は、マンホールの規格により定まる「外径」と「マンホールの中心」をもとに推定してよい。「マンホールの中心」とは、原典資料から入手できるマンホール本体又はマンホールの蓋の中心を指す。
