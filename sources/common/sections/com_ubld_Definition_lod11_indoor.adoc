|LOD1.1
|■
|ClosureSurface
|lod1MultiSurface
|GM_MultiSurface
|
|
* 床面（FloorSurface）、天井面（CeilingSurface）及び内壁面（InteriorWallSurface）を区切る仮想的な境界線に囲まれた水平面又は垂直面を取得する。
* 水平面として取得する場合、 床面の高さ又は天井面の高さのいずれか一方の一律の高さとする。
* 垂直面として取得する場合、各頂点は床面の高さ又は天井面の高さのいずれかの高さとする。
|いずれか1種類以上の境界面を区分する。
