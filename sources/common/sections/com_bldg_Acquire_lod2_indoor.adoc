h| 説明
| 部屋の形状を立体として取得し、全ての境界面を地物に区分する。 +
付属物は取得しない。 +
開口部の境界面を取得は任意となる。 +
取得例の場合は、境界面の区分のみを行い、開口部の境界面は区分していない。
| 部屋の形状を立体として取得し、全ての境界面を地物に区分する。 +
建築物内部に存在する付属物のうち、階段、スロープ、エスカレータ、輸送設備（エレベータ、エスカレータ、動く歩道）、柱、デッキ、ステージを取得する。 +
開口部の境界面を取得することができる。
| 部屋の形状を立体として取得し、全ての境界面を地物に区分する。 +
建築物内部に存在する付属物のうち、階段、スロープ、エスカレータ、輸送設備（エレベータ、エスカレータ、動く歩道）、柱、デッキ、ステージ、手すり、パネル、梁を取得する。 +
開口部の境界面を取得することができる。家具を取得することができる。
