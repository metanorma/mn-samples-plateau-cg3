[[fn_tree_model_simplify]]
[NOTE]
--
簡略化した立体とは、楕円体、球体、円錐、角錐、角柱、円柱などの単純な立体図形とする。
--

[[fn_tree_height]]
[NOTE]
--
一定高さごとに樹冠の横断面を作成し、この頂点を結び外形を構成する（樹冠内部の主枝等の表現は行わない。）が、ユースケースの必要に応じて詳細化してよい。
--
