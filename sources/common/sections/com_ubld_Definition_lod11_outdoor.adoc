|LOD1.1
|●
|UndergroundBuilding
|lod1Solid
|GM_Solid
|
|
* 地下街の上方からの正射影の外周を取得し、地上から一律の高さで下向きに押し出した立体を作成する。
|

|LOD1.1
|〇
|RoofSurface
|lod1MultiSurface
|GM_MultiSurface
|
|
* 地下街の正射影の外周を取得し、上向き面を屋根面（RoofSurface）とする。
* 外周を構成する各頂点に、地表の高さを与える。
|地表の高さは、地下街の上からの正射影の外周と地表面との交線の最下部点の高さ（最も低い標高）とする。

|LOD1.1
|〇
|WallSurface
|lod1MultiSurface
|GM_MultiSurface
|
|
* 外壁の正射影の外周を取得する。
* 外周を構成する各頂点は底面の高さ又は屋根面の高さのいずれかの高さとする。
|

|LOD1.1
|〇
|GroundSurface
|lod1MultiSurface
|GM_MultiSurface
|
|
* 地下街の上方からの正射影の外周を取得する。
* 外周を構成する各頂点に、地下街の最下端の高さを与える。
|

|lod1MultiSurface
|GM_MultiSurface

* 垂直面として取得する場合、各頂点は底面の高さ又は屋根面の高さのいずれかの高さとする。
