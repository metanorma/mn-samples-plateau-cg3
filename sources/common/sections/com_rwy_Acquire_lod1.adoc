h| 説明
| 普通鉄道、地下鉄、路面鉄道及び特殊軌道の場合は、レールの内側の領域を面をTrafficAreaとする。 +
軌道が分岐・合流する場所で地物を区切る。 分岐・合流する場所では面が重複する。 +
高さは0とする。
granularityがwayの場合は、
同一路線に上下線などの複数の軌道が存在する場合、路線ごとにまとめて一つの地物とし、granularityがlaneの場合は、軌道ごとに一つの地物とする。 +
交通空間（TrafficSpace）はTrafficAreaを底面とし、底面から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは建築限界の最高高さを原則とする。 +
交通補助領域（AuxiliaryTrafficArea）は、道床外の鉄道用地の領域を面として取得できる。 +
交通補助空間（AuxiliaryTrafficSpace）はAuxiliaryTrafficAreaを底面とし、から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは建築限界の最高高さを原則とする。 +
さらに、離隔空間（ClearanceSpace）は建築限界の上面を底面とし、底面から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは無人航空機が飛行可能な高さを原則とする。

| モノレールの場合は、軌道桁の外周に囲まれた面をTrafficAreaとする。 +
軌道が分岐・合流する場所で地物を区切る。 分岐・合流する場所では面が重複する。 +
高さは0とする。
granularityがwayの場合は、同一路線に上下線などの複数の軌道が存在する場合、路線ごとにまとめて一つの地物とし、granularityがlaneの場合は、軌道ごとに一つの地物とする。 +
交通空間（TrafficSpace）はTrafficAreaを底面とし、底面から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは建築限界の最高高さを原則とする。 +
離隔空間（ClearanceSpace）は建築限界の上面を底面とし、底面から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは無人航空機が飛行可能な高さを原則とする。 +
交通補助領域（AuxiliaryTrafficArea）及び交通補助空間（AuxiliaryTrafficSpace）は、取得対象が存在しない。 

| 索道の場合は、起点及び終点が同一となる索道のケーブルに囲まれた範囲を面をTrafficAreaとする。 +
高さは0とする。 +
granularityがwayの場合は、同一路線に上下線などの複数の軌道が存在する場合、路線ごとにまとめて一つの地物とし、granularityがlaneの場合は、軌道ごとに一つの地物とする。 +
交通空間（TrafficSpace）はTrafficAreaを底面とし、底面から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは建築限界の最高高さを原則とする。 +
離隔空間（ClearanceSpace）は建築限界の上面を底面とし、底面から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは無人航空機が飛行可能な高さを原則とする。 +
交通補助領域（AuxiliaryTrafficArea）及び交通補助空間（AuxiliaryTrafficSpace）は、取得対象が存在しない。

3+| レールに囲まれた範囲及び道床を取得する。高さは0とする。 +
レールに囲まれた範囲の形状はLOD1と同様である。 +
道床は外周により囲まれた範囲をTrafficAreaとして取得する。道床はレールに囲まれた範囲を包含する。高さは0とする。 +
いずれの地物の場合も、granularityがwayの場合は、同一路線に上下線などの複数の軌道が存在する場合は、路線ごとにまとめて一つの地物とし、granularityがlaneの場合は、軌道ごとに一つの地物とする。 +
交通空間（TrafficSpace）はTrafficAreaを底面とし、底面から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは建築限界の最高高さを原則とする。 +
交通補助領域（AuxiliaryTrafficArea）は、道床外の鉄道用地の領域を面として取得できる。 +
交通補助空間（AuxiliaryTrafficSpace）はAuxiliaryTrafficAreaを底面とし、底面から一律の高さで上向きに押し出した立体として表現できる。このとき、一律の高さは建築限界の最高高さを原則とする。 +
