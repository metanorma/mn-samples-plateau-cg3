.交通（広場）モデル（LOD2）の取得イメージ（広場内の区分）
|===

| 車道部、車道交差部、島及び歩道部を区分する。
| LOD2.0（granularity=way）の区分を細分する。 +

| LOD2.0（granularity=lane）の区分を細分する。 +
歩道部のうち、植栽を区分する。
| LOD2.1（granularity=lane）の区分を細分する。細分はユースケースに応じて決定する。
